# Geek Social

A simple `Fullstack` social app project that allow users to create their own "Playlist" of TV shows, Movies, Animes, Games, etc.
Below are the main features:

- Authentication
- Create Playlist
- Follow playlists
- Follow users
- Create posts
- Like posts
- Comment posts
- Edit profile
- Chat
